Simple Calculator (Java)

A simple command-line calculator built in Java that performs basic arithmetic operations: Addition, Subtraction, Multiplication, and Division.
📌 Features
➕ Addition
➖ Subtraction
✖️ Multiplication
➗ Division
🚫 Division by zero handling
✅ User-friendly console interface
🔢 Supports decimal numbers using double

📂 Project Structure
Simplecalculator.java
🚀 Getting Started
Prerequisites
Java JDK 8 or higher
Any Java IDE (VS Code, IntelliJ IDEA, Eclipse) or Command Prompt
Compile the Program
javac Simplecalculator.java
Run the Program
java Simplecalculator
